> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` può essere usato solo in un file `.astro`. `import.meta.glob(GLOB_STR)` può essere usato per ottenere un risultato simile.

## Cosa è andato storto?
`Astro.glob()` può essere usato solo in un file `.astro`. Puoi usare [`import.meta.glob()`](https://vitejs.dev/guide/features.html#glob-import) per ottenere lo stesso risultato.
