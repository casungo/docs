> **CantRenderPage**: Astro non riesce a trovare alcun contenuto da renderizzare per questo percorso. Non c'è alcun file o redirect associato a questo percorso.

## Cosa è andato storto?
Astro non è riuscito a trovare un file associato con contenuto mentre cercava di renderizzare il percorso. Questo è un errore di Astro e non un errore dell'utente. Se il riavvio del server di sviluppo non risolve il problema, si prega di segnalare il problema.
