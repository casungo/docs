:::caution[Deprecato]
Le collezioni che non esistono non generano più un errore. Viene invece fornito un avviso.
:::

> Una collezione chiamata tramite `getCollection()` non esiste.

## Cosa è andato storto?
Quando una collezione viene chiamata, assicurarsi che esista una directory di collezione con il nome richiesto dentro `src/content/`.
