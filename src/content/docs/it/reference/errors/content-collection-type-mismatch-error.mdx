> **ContentCollectionTypeMismatchError**: COLLECTION contiene voci EXPECTED_TYPE, ma è configurata come una collezione ACTUAL_TYPE.

## Cosa è andato storto?
Le collezioni di contenuti devono contenere voci del tipo configurato. Le collezioni sono `type: 'content'` per impostazione predefinita. Prova ad aggiungere `type: 'data'` alla configurazione della tua collezione per le collezioni di dati.
